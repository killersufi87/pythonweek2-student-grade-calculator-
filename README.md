# Student Grade Calculator

## Project Overview
A Python program that calculates student grades based on marks.

## Features
- Input validation
- Grade calculation using if-elif-else
- Encouraging messages
- Error handling

## How to Run
1. Install Python
2. Open terminal
3. Run: python grade_calculator.py

## Grading Logic
A: 90-100
B: 80-89
C: 70-79
D: 60-69
F: Below 60
